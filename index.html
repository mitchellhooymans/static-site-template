Build Something Amazing

A clean, modern static-site skeleton designed for GitHub Pages. Perfect for academic portfolios, research groups, or project docs.

Explore Features View on GitHub

Key Highlights

Everything you need to get started — no build tools, no frameworks, just clean code.

Lightning Fast

Pure HTML, CSS, and JavaScript — no build step, no dependencies. Open index.html and you're live.

Easy Theming

Change colours, fonts, and spacing in one file — variables.css. The entire site updates instantly.

Reusable Components

The navbar and footer live in separate files. Edit once, and every page updates — powered by a tiny JS loader.
